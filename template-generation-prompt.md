# NyxAI 模板批量生成 Prompt

## 任務
請根據以下範例和規範，生成 **20 個**高質量成人小說模板。

---

## 📋 範例模板（青梅竹馬）

```json
{
  "id": "classic-002",
  "slug": "childhood-sweetheart",
  "name": "青梅竹馬",
  "category": "classic",
  "description": "那個從小一起長大的女孩，什麼時候變得這麼迷人了？",
  "baseScenario": "你們是住在隔壁的青梅竹馬，從小一起上下學。她總是大大咧咧地穿著睡衣來你家蹭飯，從不把你當外人。但最近你發現，這個從小看到大的女孩，身材越來越成熟，笑容也越來越讓人心跳加速...",
  "writingStyle": "日常溫馨中帶有微妙張力，注重細節描寫（睡衣領口、髮香、不經意的肢體接觸）",

  "pace": "slow",
  "intensity": "mild"

```

## 🎯 輸出格式要求

每個模板必須包含以下字段：

| 字段 | 類型 | 說明 |
|------|------|------|
| id | string | 格式：`{category}-{三位數}`，如 classic-003 |
| slug | string | URL友好，英文小寫，用連字符，如 office-secret |
| name | string | 中文名稱，2-6個字 |
| category | string | 分類：classic/campus/mature/career/taboo/ntr |
| description | string | 描述，20-30字，吸引人 |
| baseScenario | string | 基礎情境，100-150字，具體生動 |
| writingStyle | string | 寫作風格，50-80字，指定描寫手法 |
| atmosphere | string | 氛圍，30-50字，情感基調 |
| pace | string | 節奏：slow/medium/fast |
| intensity | string | 強度：mild/moderate/intense |

## 📁 分類說明

### classic（經典）- 5個
初戀、重逢、青梅竹馬、同學會、前女友等經典情感題材

### campus（校園）- 4個
學姐、學妹、女老師、校花、圖書館、宿舍等校園場景

### mature（人妻）- 4個
鄰居太太、少婦、寂寞主婦、閨蜜等成熟女性題材

### career（職場）- 4個
女上司、秘書、護士、空姐、健身房教練等職業幻想

### taboo（禁忌）- 2個
年齡差、秘密戀情、身份禁忌等背德題材

### ntr（NTR）- 1個
出軌、復仇、綠帽等NTR題材

## ✨ 質量要求

### baseScenario（基礎情境）
- 具體場景設定（時間、地點、關係）
- 包含一個「轉折點」或「張力來源」
- 讓讀者能想像畫面

**好例子**：
> 「你們是住在隔壁的青梅竹馬...但最近你發現，這個從小看到大的女孩，身材越來越成熟」

**壞例子**：
> 「一個男生和一個女生是鄰居，他們從小認識」

### writingStyle（寫作風格）
- 指定具體描寫手法
- 可包含：心理描寫、感官細節、對話風格、敘事節奏

**好例子**：
> 「日常溫馨中帶有微妙張力，注重細節描寫（睡衣領口、髮香、不經意的肢體接觸）」

### atmosphere（氛圍）
- 情感基調（青澀、禁忌、刺激、溫馨等）
- 可加入心理層面描述

**好例子**：
> 「青澀曖昧，帶有「太熟了反而不好下手」的糾結」
