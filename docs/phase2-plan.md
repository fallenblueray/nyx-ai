# Phase 2 病毒增長系統 - 詳細規劃

## 核心改進（基於反饋）

### 1. 故事 ID 系統（新增）
- URL: `/story/{story_id}`
- 內容：故事標題 + 預覽 + 完整故事 + 分享卡 + 「用同模板生成」CTA
- 功能：SEO + 分享入口

### 2. 排行榜娛樂化（改進）
- 今日最熱門 (view + share)
- 本週最瘋狂 (AI 判定情節強度)
- 最多分享 (share_count)
- 最多閱讀 (view_count)

### 3. 隨機故事一鍵生成（改進）
- 流程：點擊 → 直接生成故事（無需填表）
- 技術：前端調用 generate-story API

### 4. 病毒題材池（新增）
- 預設組合：題材 + 場景 + 開場
- 確保組合具有病毒傳播潛力
