# V0 Landing Page 提示詞

## 模板參考
使用模板：https://v0.app/templates/hero-section-with-react-bits-floating-lines-yih2hMiv5q3

## 專案背景
**NyxAI** - AI 故事生成平台，「五秒開展故事」，專為創作小黃文設計

## 設計要求

### 色彩系統
- **主色調**：深藍到紫藍的漸變風格
- **Primary**：`#1e3a8a` (深藍) → `#7c3aed` (紫) 漸變
- **Accent**：`#3b82f6` (亮藍) / `#8b5cf6` (亮紫)
- **Background**：`#0f172a` (深 slate) 或 `#020617` (更深)
- **Text**：`#f8fafc` (白) / `#cbd5e1` (灰)

### Hero Section 內容
```
標題：NyxAI
副標：五秒開展故事
描述：AI 驅動的極致創作體驗。輸入一句話，立即生成專業級故事。
CTA：開始創作
```

### 視覺效果
- **Floating Lines**：保留模板的 floating lines 動畫
- **漸變背景**：深藍到紫藍的動態漸變
- **Glow Effect**：按鈕和重點文字添加藍紫色光暈
- **粒子效果**：可選的微妙粒子動畫

### 響應式設計
- Desktop：完整動畫和效果
- Tablet：簡化動畫
- Mobile：保留漸變，簡化 floating lines

## 技術要求
- React + TypeScript
- Tailwind CSS
- Framer Motion（動畫）
- 可複用的組件結構

## 輸出
生成完整的 Hero Section 代碼，包含：
1. 主組件文件
2. 動畫配置文件
3. 顏色主題配置
4. 使用示例
